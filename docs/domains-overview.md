# domains/

本目录每个一级子目录都是一个**业务域（bounded context）**。业务域承载业
务规则、领域语言、数据形态。流水线阶段是业务域内部的子模块，不是独立的
业务域。

| 业务域 | 状态 | 说明 |
|---|---|---|
| `gov_policy/` | 第一阶段建设中 | 政府产业政策（中央 + 部委 + 31 省市） |
| `exchange_policy/` | 未启动 | 金融交易所与行业协会政策（计划） |
| `oversea_policy/` | 未启动 | 境外监管政策（涉跨境合规分支，计划） |

新增业务域时，先在 `docs/prod-spec/` 写业务规格再起目录；不要在 domains 下
建仅承载技术能力的目录（那是 `infra/` 的事）。
